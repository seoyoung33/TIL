em적용 자식 글자
em은 부모의 글자크기값을 상대적으로 인식해서 최종 크기를 정하는 크기 단위이며 rem은 부모의 글자크기 관계없이 자기 자신에 설정한 크기값만 인식하는 글자 크기 단위이다. 모바일부터 PC까지 모든 디바이스를 상대적으로 인식하며 글자 크기 계산이 쉽다는 특징 때문에 실제 개발시에는 대부분 rem 단위를 사용한다.

line-height는 피그마에서부터 % 단위를 사용하여 디자인하고 CSS 진행시에 피그마 단위를 그대로 사용하여 %를 입력하거나 약자를 활용하여 예를 들어 150% 라면 1.5라고 작성하는 식으로 행간을 적용한다.
